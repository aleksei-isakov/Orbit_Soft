线上广告自动化购买和管理

Orbit 广告服务器

用于展示、上下文和视频广告管理的服务器

  • 可创建响应时间极短的可扩容高负荷容错解决方案

  • 用其可在网站和应用程序中管理广告

  • 可通过 API 和 SDK 进行整合

Orbit DSP

用于在实时拍卖会上购买广告位的自动化系统

  • 可借助于动态竞买系统和 RTB 技术影响交易结果

  • 可通过灵活的管理（如广告位应价、受众定向、系列活动时间安排、预算、竞买的管理）提高广告主的收入

Orbit DMP

数据管理平台

  • 存储、整理和分析数据

  • 根据指定标准确定目标受众

  • 收集并处理用户和用户活动相关数据

  • 根据某些规则细分用户群体。

  • 通过外部系统对用户进行同步和身份验证

Orbit SSP

自动化存货销售系统

  • 用其可通过 RTB 技术销售广告位

  • 可通过适用于 Android 和 iOS 操作系统的 API 软件协议和 SDK 开展工作。

预报和预测活动

预测系统

  • 预报和预测系统

  • 根据活动和用户行为预测用户的行动。此系统根据熵值法和线性回归法训练并构建模型

视频推荐

  • 推荐系统

  • 根据行动和历史数据推荐按用户的兴趣定制的内容
